# Security

TribeWarez takes security seriously. If you believe you have found a security vulnerability, please report it responsibly.

## Reporting

- **Do not** open a public issue.
- Open a private security advisory on GitHub: **Security** tab → **Advisory** → **Report a vulnerability** for this repository.

## Scope

- Bugs that could lead to device compromise, credential leakage, or RPC/manufacturing abuse are in scope.
- Please include steps to reproduce and impact assessment where possible.

## Acknowledgments

We will acknowledge responsible disclosure once the issue is assessed.
