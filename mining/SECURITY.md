# Security

TribeWarez takes security seriously. If you believe you have found a security vulnerability, please report it responsibly.

## Reporting

- **Do not** open a public issue.
- Email security concerns to the maintainers or open a private security advisory on GitHub: **Security** tab → **Advisory** → **Report a vulnerability** for this repository.

## Scope

- Bugs that could lead to consensus failure, fund loss, or compromise of validator/miner or dependent systems are in scope.
- Please include steps to reproduce and impact assessment where possible.

## Acknowledgments

We will acknowledge responsible disclosure once the issue is assessed.
